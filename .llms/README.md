# .llms Directory

This directory contains documentation and context for LLM (Large Language Model) agents working on this codebase.

## Contents

- [ARCHITECTURE.md](./ARCHITECTURE.md) - Overall system architecture
- [INTEGRATIONS.md](./INTEGRATIONS.md) - Integration adapters documentation
- [CONSTANTS.md](./CONSTANTS.md) - Frontend constants and frozen dictionaries
- [TESTING.md](./TESTING.md) - Testing patterns and guidelines
- [CONVENTIONS.md](./CONVENTIONS.md) - Code conventions and patterns

## Purpose

These files provide context for AI coding assistants to better understand:
- The project structure and patterns
- How to add new features consistently
- Testing requirements
- Code style conventions

## Usage

When working with an LLM agent, reference these files to:
1. Provide context about the project
2. Ensure consistency with existing patterns
3. Understand the integration architecture
