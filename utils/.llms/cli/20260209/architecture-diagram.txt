## Component Responsibilities

### ClientStatisticsDashboard
- **What:** Orchestrates all statistics sections and filtering
- **Data:** Receives raw clients/projects/leads, calculates all stats
- **Events:** Emits `select-client` when user wants to see details
- **UI:** Filter chips, 6 dashboard sections, highlights button

### ClientHighlightsModal
- **What:** Modal showing top 10 high-potential clients
- **Data:** Receives clients/projects/leads, calculates scores
- **Events:** Emits `select-client` to jump to client in table
- **UI:** Ranked cards with scores, reasons, and "Ver Detalhes" button

### ClientDetailView
- **What:** Inline expandable detail panel in table row
- **Data:** Single client + filtered projects/leads
- **Events:** Emits `close` when user dismisses panel
- **UI:** Contact info, timeline, projects list, lead stats, notes

### useClientQuery (Composable)
- **What:** Safe URL state management for expanded client
- **Data:** Reactive `selectedClientId` synced with URL
- **Functions:** `setClientQuery()`, `clearClientQuery()`, validation
- **Security:** Regex validation prevents injection

## Performance Considerations

- **Lazy Rendering:** Detail views only mount when expanded
- **Computed Properties:** Stats calculated once, cached by Vue
- **Conditional Sections:** Hidden sections don't render charts
- **Event Delegation:** Single click handler for all expand buttons
- **Smooth Scrolling:** Native CSS `scroll-behavior: smooth`
